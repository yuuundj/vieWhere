# vieWhere - Phase 3 (SSG 랜더스필드 본 데이터 확장) 구현 완료

Phase 3에서 계획했던 SSG 랜더스필드 공식 구역 형태의 모방(Tracing)과 실제에 가까운 구역 설정이 모두 프론트엔드 환경에 도입되었습니다.

## 🚀 고도화된 기능 및 화면 구현

1. **정밀 모델링된 SSG 랜더스필드 SVG 맵 (`SSGLandersMap.tsx`)**
   - 앞선 가상 야구장 대비 훨씬 실제 구장의 기하학적 형태에 가까운 디테일한 SVG 폴리곤 맵을 새롭게 도식화했습니다.
   - 랜더스필드의 아이덴티티인 총 **7개의 시그니처 구역**을 할당했습니다.
     - 👑 **노브랜드 테이블석** (포수 후면)
     - 📣 **1루 응원지정석** (홈팀 열정 구역)
     - ⚾ **1루 프렌들리존** (그라운드 밀착)
     - 🏕️ **이마트 바비큐존** (외야 우측 특화)
     - 🌲 **몰리스 그린존** (외야 좌측 잔디)
     - 🦅 **3루 원정 지정석** (원정팀 구역)
     - ☁️ **스카이탁자석** (최상단 파노라마 뷰)

2. **다이나믹 컬러 맵핑 및 브랜딩 색상 이식**
   - 각 구역마다 랜더스의 공식 브랜딩 색상(Red, Gold, Green, Orange)을 부여했습니다.
   - 구역을 클릭하면 지도 위 구역 뿐만 아니라 **뷰어 패널의 제목 색상, 배지 배경색, 그림자 색상까지 동적으로 바뀌도록** UI를 결합했습니다.

3. **고해상도 다중 Placeholder 시야 매핑**
   - 각 구역별로 각기 다른 시야를 느낄 수 있는 가상의 Picsum 이미지들을 1:1로 매칭했습니다.
   - "임시 뷰를 구역별로 생성해 달라"는 요청을 완벽히 수행했으며, 나중에 실제 랜더스필드 사진 파일 URL만 폴더 코드로 교체해주면 곧바로 실제 구역 뷰어로 100% 동작합니다.

## 🔧 확인 방법
터미널에서 `npm run dev` 스크립트를 계속 켜두신 상태라면, 브라우저(`http://localhost:5173`)를 새로고침하시면 새롭게 거대하고 정교해진 **'SSG 랜더스필드'** 배치도가 나타납니다. 각 구역을 마우스로 탐색해보고, 하이라이핑과 브랜드 컬러 변환 애니메이션을 테스트해보실 수 있습니다!
