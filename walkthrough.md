# vieWhere - Phase 2 (가상 야구장 맵 테스트) 구현 완료

Phase 2에서 목표했던 가상의 야구장 스태디움 인터랙션 기능이 프론트엔드 환경에 도입되었습니다.

## 🚀 추가된 기능 및 화면 구현

1. **가상 야구장 인터랙티브 SVG (Virtual Stadium Map)**
   - 경기장(내야/외야/잔디) 형태를 구조화한 SVG를 도식화했습니다.
   - **VIP석, 1루 응원석, 3루 지정석, 외야 일반석** 총 4개의 가상 구역이 클릭과 Hover(마우스 오버)가 가능하도록 구현되었습니다.
   - 마우스를 올릴 때마다 구역이 하이라이팅되며, 클릭 시 강렬한 색상으로 고정됩니다.

2. **구역 상세 뷰어 패널 (Sector View Panel)**
   - 오른쪽 또는 하단 패널에 위치하는 컴포넌트입니다.
   - 구역을 클릭하면, 해당 구역의 **예상 시야 사진(Unsplash 테스터 이미지 활용), 구역 설명, 예상 가격 정보**가 부드럽게 등장(Fade-in)합니다.
   - 사진 위에 효과적인 Glassmorphism 배지를 달고 프리미엄 디자인 토큰을 활용했습니다.

3. **반응형 웹 UI 레이아웃**
   - 데스크톱에서는 맵과 패널이 좌우 2분할(Grid 구성)로 쾌적하게 보입니다.
   - 모바일 크기로 브라우저를 줄이면 자동으로 상하단(Column 구성)으로 정렬되며 보기 좋게 압축됩니다.

## 🔧 확인 방법
터미널에서 `npm run dev` 스크립트를 계속 켜두신 상태라면, 브라우저(`http://localhost:5173`)를 새로고침하시면 완성된 가상 인터랙티브 맵 화면을 즉각적으로 확인하실 수 있습니다. 맵 안의 각 구역 텍스트를 클릭해 보세요!
